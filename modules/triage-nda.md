# Режим: Скрининг NDA

Быстрая классификация NDA по светофору: 🟢 можно подписать / 🟡 нужна проверка юриста / 🔴 не подписывать.

## ШАГ 0 - Приватность (обязательно первым)
Проверь, **не запрещает ли сам NDA раскрытие его текста третьим сервисам/обработчикам.** Если запрещает - предупреди: полный разбор такого NDA сторонним ИИ может нарушать этот же NDA. Дальше работай по обезличенному пересказу условий, а не по полному тексту.

## Критерии проверки (10+)
Пройди по каждому, помечая 🟢/🟡/🔴:
1. **Структура** - взаимное или одностороннее? (Односторонее не в твою пользу - флаг, если ты тоже раскрываешь.)
2. **Объём определения конфиденциальной информации** - не слишком ли широкий? («Вся информация» без границ - 🟡/🔴.)
3. **Стандартные исключения** - есть ли: общедоступное, предшествующее владение, независимая разработка, законно полученное от третьих лиц? (Отсутствие - флаг.)
4. **Разрешённые раскрытия** - по закону/суду, аффилиатам, советникам с обязательством конфиденциальности.
5. **Срок действия и период сохранения** - **бессрочность = 🔴 красный флаг.** Разумный период (напр. 2-5 лет) - норма.
6. **Скрытые условия** - несолицитация, неконкуренция, эксклюзивность, уступка ИС, standstill. Это не функции NDA - их наличие 🟡/🔴.
7. **Применимое право и юрисдикция** - удобны ли для тебя? (Экзотическая юрисдикция - флаг.) Сверь по `reference/` нужной страны.
8. **Возврат/уничтожение** информации по запросу/окончании.
9. **Средства защиты** - судебный запрет (injunction), неустойка - не чрезмерны ли.
10. **Обязательства по защите** - стандарт хранения, уведомление об утечке.

## Вывод
- **Один итоговый цвет** 🟢/🟡/🔴.
- **3-5 ключевых причин** этого цвета (сначала самое важное).
- **Действие:** подписать сегодня / отправить юристу / не подписывать и запросить правки.
- JIT-сверка правовых пунктов, двойная маркировка, эскалация при триггерах.

## Частые контексты
- Входящий NDA от потенциального партнёра - можно ли подписать сегодня?
- NDA вендора перед демо - безопасно ли?
- Взаимный NDA от конкурента - проверить скрытые несолицитацию/неконкуренцию/standstill.
- NDA инвестора перед due diligence - не слишком ли широка конфиденциальность, нет ли уступки ИС.
- Пакет из нескольких NDA - рассортируй: что подписать сразу, что проверить, на что возразить.
