# Режим: Проверка соответствия

Опиши, что планируется - скилл проверит на соответствие GDPR / EU AI Act / ePrivacy / праву Германии / закону Украины. Выявит применимые нормы, риски, нужные согласования и что исправить до запуска.

## Перед стартом
Уточни (если не ясно из запроса): **что** делаем, **с какими данными**, **чьими** (откуда пользователи - ЕС / Германия / Украина / третьи страны).

## Процесс
1. **Определи применимые нормы** и открой соответствующие `reference/`:
   - Персональные данные → `gdpr.md` (всегда) + `ukraine.md`, если есть украинские субъекты.
   - ИИ / ML / автоматизированные решения → `eu-ai-act.md`.
   - Cookies / рассылки / трекинг / маркетинг → `eprivacy.md`.
   - Локальная специфика → `germany.md` / `ukraine.md`.
2. **По каждой норме:** применима ли? какие требования? что нужно сделать **до запуска**?
3. **JIT-сверка** всех сроков/статей/порогов. **Двойная маркировка** (норма / применение).

## Вывод (структура)
- **Применимые законы** - перечень с короткой причиной.
- **Риски** - что нарушается/под угрозой, со светофором.
- **Необходимые документы/согласования** - DPA (GDPR ст. 28), DPIA (ст. 35), согласие, cookie-баннер, политика конфиденциальности, маркировка ИИ-контента и т.д.
- **Что исправить до запуска** - конкретный чек-лист.
- **Светофор готовности** к запуску + эскалация при критике.

## Типовые сценарии
- **Новая фича продукта** (напр. биометрия/Face ID) - спецкатегории (GDPR ст. 9), DPIA, основание.
- **Маркетинг с отзывами и фото клиентов** - согласие на образ, основание, право на отзыв согласия.
- **Email-база через лид-магнит** - opt-in (ePrivacy), разделение согласий, отписка.
- **ML-модель на переписке поддержки** - основание, минимизация, спецкатегории, EU AI Act (прозрачность чат-бота).
- **Трансфер данных за пределы ЕС** - механизм (адекватность / SCC / BCR), Schrems II-меры.
- **Мониторинг сотрудников** (экран, активность) - данные работников (BDSG §26 для Германии - `[verify@use]`), пропорциональность, информирование.
- **Данные несовершеннолетних** - возраст согласия, родительское согласие, повышенная защита.
- **Реферальная программа** - обработка данных приглашённых, основание, антиспам.
