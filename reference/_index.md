# Навигатор правовой базы

Карта юрисдикций. Открывай нужный файл, дальше работай по JIT-протоколу из `SKILL.md`.

| Тема / юрисдикция | Файл | Статус базы |
|---|---|---|
| Защита данных ЕС (стержень) | `gdpr.md` | ✓ ключевые параметры выверены (эталон) |
| EU AI Act (регулирование ИИ) | `eu-ai-act.md` | каркас + якоря, `[verify@use]` |
| ePrivacy (cookies, e-коммуникации) | `eprivacy.md` | каркас + якоря, `[verify@use]` |
| Право Германии (договоры, аренда, труд, данные) | `germany.md` | каркас + якоря, `[verify@use]` |
| Право Украины (данные, договоры) | `ukraine.md` | каркас + якоря, `[verify@use]` |
| Шкалы оценки (светофор, матрица) | `frameworks.md` | стабильно, web не нужен |

**Принцип выбора:**
- Любой вопрос про персональные данные в ЕС/с лицами из ЕС → начинай с `gdpr.md`.
- ИИ-система, ML-модель, автоматизированные решения → `eu-ai-act.md` (+ gdpr для данных).
- Cookies, рассылки, трекинг, маркетинг → `eprivacy.md` (+ gdpr).
- Немецкий договор/аренда/трудовой/местная обработка → `germany.md` (+ gdpr для данных).
- Украинский контрагент/обработка данных в Украине → `ukraine.md`.

Документ часто задевает несколько баз сразу (напр. немецкий SaaS-договор с обработкой данных = `germany.md` + `gdpr.md` + возможно `eu-ai-act.md`). Открывай все релевантные.
