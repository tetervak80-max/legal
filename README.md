# legal

Claude Code / Claude Agent **Skill** - личный ИИ-юрист для права ЕС и Украины.

> ⚠️ Это инструмент-помощник, а не замена живого юриста. Скилл сам по себе маркирует уверенность и эскалирует к человеку-юристу в спорных местах. Не используйте как окончательную юридическую консультацию.

## Шесть режимов

| Режим | Что делает |
|---|---|
| **review-contract** | Постатейная проверка договоров с редлайнами |
| **triage-nda** | Скрининг NDA по принципу «светофора» |
| **compliance-check** | Проверка на соответствие: GDPR / EU AI Act / ePrivacy / право Германии / закон Украины о защите данных |
| **risk-assessment** | Оценка юридических рисков по матрице (вероятность × тяжесть) |
| **response** | Юридические письма: запрос субъекта данных, возражения по условиям, отказ от услуги, ответ на претензию |
| **brief** | Юридические брифы: исследование по теме, бриф по инциденту |

## Когда срабатывает

Триггеры: «проверь договор», «проверь NDA», «оцени риски», «GDPR», «соответствие», «редлайн», «юридическое письмо», «ответ на претензию». Также - когда загружаешь договор/NDA/оферту/DPA/претензию и просишь проверить.

Не используется для не-юридических текстов (статьи, посты, маркетинг).

## Структура

- `SKILL.md` - роутер режимов и общие правила (маркировка уверенности, эскалация).
- `modules/` - инструкции под каждый режим.
- `reference/` - справочники по GDPR, EU AI Act, ePrivacy, праву Германии и Украины, общие риск-фреймворки.

## Лицензия

Свободно используй и адаптируй. Без гарантий; не является юридической консультацией.
